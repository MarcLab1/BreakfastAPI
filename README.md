# The BreakfastAPI Android App

## Concepts 

  * Kotlin
  * Jetpack compose
  * Coroutines 
  * MVVM
  * Lifecycle scopes
  * Retrofit
  * okHttp
  * Hilt (Dagger)
  * Coil
  * Compose Navigation
  * Room persistence library
  * Scaffold
  * Snackbar/SnackbarHost
